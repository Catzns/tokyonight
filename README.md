Nobody in their right mind should fall in love with a text editor color scheme, and yet I have.

Tokyo Night is a dark, low-contrast, pale color scheme inspired by nighttime lights of downtown Tokyo, created by [Enkia](https://github.com/tokyo-night/tokyo-night-vscode-theme) and extended by [folke](https://github.com/folke/tokyonight.nvim). This repository will house the ports I make of Tokyo Night to applications where one does not already exist.
